CNS*1995 Macros

A TeX/LaTeX macro package is now available for formatting papers for the CNS*95 proceedings. This package was written by John Pezaris at Caltech (pz@caltech.edu), and although every effort has been made to insure that it produces the correct format, if you chose to use the package, you should check your paper yourself to verify it meets the submission guidelines. Copies of the submission guidelines are available from Judy Macias, Division of Biology 216-76, Caltech, Pasadena, CA 91125.

The macro package is available via anonymous ftp from http://www.neuroinf.org/CNS/cns-arch/cns95/cns95.tex

Instructions for usage of the macro package are available in postscript form as a sample paper. The source code for this sample paper is tacked to the end of the macro package (commented out), and the postscript is available as http://www.neuroinf.org/CNS/cns-arch/cns95/sample.ps

Read the sample paper before using the macro package. All questions regarding bugs, comments, and usage of the macros should be directed to John Pezaris, and NOT to the CNS*95 organizers.
