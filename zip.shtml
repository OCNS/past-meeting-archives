Instructions for creating a flat zip-file.

Windows:

In case WinZip is not installed on your PC you can download the
program from http://www.winzip.com

  • Start WinZip and choose "WinZip Classic" from the startup-screen.
  • Press "New" to create a new zip-file (archive).
  • In the "New" window that appears you can select a directory ("My
    Documents" for example) to store the zip-file.
  • In the field "File name" you provide the name of the new zip-file
    (you do not need to provide the file-extension zip, the program
    will do this for you).
  • Press then the "OK" button
    In the "Add" window that now appears you select the files you want
    to put in the zip-file (you can make multiple selections if you
    wish)
  • Make sure the following is set to:
    Action : Add (and replace) files
    Compression : Normal
    Store filenames in 8.3 format : unchecked
    Save full path info : unchecked
  • The press the "Add" button.
  • In the main window you see a list of the files which are added to
    the zip-file, if needed you can always press the Add button to add
    more files.
  • Select "Exit" from the menu item "File" to close the zipfile and
    exit the program.

Unix / Linux:

It's a good idea to move/copy all the files for the zip-file to one
directory.

  • cd to that directory
  • type : zip name_of_the_zip *
    This command will create the zip-file "name_of_the_zip.zip" in the
    current directory and adds all the files (except the file
    "name_of_the_zip.zip") in found the current directory to the
    zip-file.
  • or type : zip name_of_the_zip file1 file2 file3 ...
    This command will create the zip-file "name_of_the_zip.zip" in the
    current directory and adds only file1 file2 file3 ... to the
    zip-file.
