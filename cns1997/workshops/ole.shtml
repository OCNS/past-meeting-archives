PHYSIOLOGICALLY REALISTIC MEMORY NETWORKS.

Ole Jensen

Physiologically realistic principles are rarely taking to serious when synaptic encoding of memory is studied in neural networks. We will address the issues 1) how does the time window of LTP with respect to pre- and post-synaptic activity affect learning. 2) LTP requires several repeated tetani. It takes about 1/2 minute for a potentiated synapse to be expressed. Does this argue for mechanism maintaining memories active prior to synaptic encoding?
